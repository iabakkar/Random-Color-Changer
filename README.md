# 🎨 Random Color Changer

A fun app that changes background color randomly.

## 🚀 Features

* 🌈 Random RGB color generation
* 🎯 One-click color change
* ⚡ Instant UI update

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 📸 Preview

Click the button and watch the background color change.

## 💡 Learning

* RGB color system 🎨
* Random numbers 🔢
* DOM styling 🧠

## ▶️ How to Use

1. Click **Change the color!**
2. Background changes randomly
